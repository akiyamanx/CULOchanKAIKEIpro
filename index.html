❓ ヘルプ

📱 基本的な使い方

📷 レシート読込

レシートを撮影すると、OCR技術で文字を読み取り、自動的に品名・金額を入力します。
読み取った内容は「経費一覧」に保存されます。

📝 見積書作成

品名マスターから材料を選んで見積書を作成できます。
利益率を設定すると自動で売価を計算します。
完成した見積書はPDFで出力できます。

📄 請求書作成

見積書をベースに請求書を作成できます。
顧客情報は「顧客管理」から呼び出せます。

📊 確定申告サポート

経費と売上を自動集計し、確定申告をサポートします。
青色申告・白色申告どちらにも対応しています。

🔗 API連携について

🤖 AI機能（Gemini API）

GoogleのGemini AIを使って、レシートの読み取り精度を大幅に向上させることができます。

✅ API連携のメリット

  • レシートの読み取り精度が大幅向上（手書き文字も読める！）
  • 品名・金額を自動で分類
  • 合計金額を自動計算
  • 勘定科目を自動判定（確定申告で便利！）

📝 APIキーの取得方法

  1. Google AI Studioにアクセス
  2. Googleアカウントでログイン
  3. 「Get API key」をクリック
  4. 生成されたAPIキーをコピー
  5. ⚙️設定画面の「Gemini API Key」に貼り付け

⚠️ APIキーは大切に管理してください。他の人に共有しないでください。
💡 APIなしでも基本的なOCR機能は使えます（Tesseract.js）

❓ よくある質問

Q. データはどこに保存されますか？

A. すべてのデータはお使いの端末（スマートフォン/PC）に保存されます。クラウドには送信されないので安心です。

Q. 機種変更する場合は？

A. ⚙️設定画面の「データのバックアップ」でJSONファイルを保存し、新しい端末で「データの復元」を行ってください。

Q. オフラインで使えますか？

A. はい！一度読み込んだ後はオフラインでも使えます（AI機能を除く）。
